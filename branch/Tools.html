Tools

Tools are items with their own functions, used manually by the player to carry out specific tasks ranging from terraforming to hauling large objects.

Player Tools
  • Terrain Tool is a tool that is given to the player when started any world. You can take the Terrain Tool by pressing the Q. Terrain Tool can be modified by Augments, attachments which alter the way the tool interacts with soil. Terrain Tool has 3 modes. Mining mode allows you to mine soil and other resources. Building mode lets you build if you have at least 1 small canister in your backpack. And the last mode is the smooth mode which lets you make smooth platforms and edges. By hovering on a ground at a certain angle, than triggering the tool, you are able to make smooth paths up to mountains, down to caves or make your base platform smooth. It works after triggering Terrain Tool it keeps the angle and than builds or mines into the angle, that you selected.
  • Creative Tool can be modified by pressing R while it is equipped. It is only available in Creative Mode.
  • Aero can be used as a free cam. It is only available in Creative Mode.

Vehicle Tools
  • Drill Strength 1, Drill Strength 2 and Drill Strength 3 can excavate and gather materials quickly, can dig through hard sedimentary materials when attached to the front of a Rover or on a Crane.
  • Winch: Can be used to pull objects when attached to a rover or (for smaller objects) held in hand. Or to anchor vehicles to the ground.
  • Crane: Can be used with a Drill to mine Resources.
  • Paver: Can be used to build soil level with the camera's orientation, especially useful in combination with a Drill.
